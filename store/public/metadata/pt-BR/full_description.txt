#### Store
#### App base para todas as lojas do VTEX IO.

Quando você instalar esta app, você terá as principais funcionalidades abaixo: 

- Estrutura de dataLayer da sua loja. 
- Todos os seus produtos com Microdata.

Para ver todas as funcionalidades que estamos trabalhando [acesse o changelog](https://github.com/vtex-apps/store/blob/master/CHANGELOG.md)

Você pode ler mais informações sobre a Store App [no README do Github.](https://github.com/vtex-apps/carousel/blob/master/README.md)
